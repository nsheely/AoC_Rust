# Advent of Code - Rust Solutions

Advent of Code solutions in Rust using [cargo-aoc](https://github.com/gobanos/cargo-aoc).

## Quick Start

```bash
# Run a solution
cargo aoc -d 1

# Benchmark
cargo aoc bench -d 1

# Or use the Makefile
make day1
make benchday1
```

## Setup

```bash
# Install cargo-aoc
cargo install cargo-aoc

# Download your input (first time only)
cargo aoc credentials <session-cookie>
cargo aoc input -d 1
```

## Notes

- See `make help` for all available commands
